### README FOR KMEANS RBF ###

Files:
    main.py: This file contains all of the code written for the KMEANS RBF Neural Network
    

Project Dependencies:
    You must have python 3.9 installed on your machine.
    You must have numpy installed.
        To install, type 'pip install numpy' in your terminal
    You must have matplotlib installed.
        To install, type 'pip install matplotlib'

Run Instructions:
    main.py:
        In your terminal, go the directory where main.py is saved and type:
        'python main.py'
